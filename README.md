# TicTac-Go

This is a tic-tac-toe game implementation made using raylib written in Golang.

-   [Raylib](https://www.raylib.com/index.html)

### Run the Game

-   Install required libraries **[Ubuntu/Linux only]**

```
bash ./requirements.sh
```

-   Install dependencies

```
go mod download
```

-   Run the game

```
make
```

-   Build executable

```
make build && make run
```
